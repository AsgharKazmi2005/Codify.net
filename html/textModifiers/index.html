Text Modifiers

HTML Text Modifiers are ways we can change our text using strictly html. Oftentimes when writing we want to bold or underline or italicize something to emphasize it. Sometimes we need to use superscripts and subscripts among other text modifiers. They are really important and on this page you will learn all of them. We can also add multiple modifiers together if we want to. Make sure to write sample code using the following elements to gain practice.

Bold & Strong Elements

In HTML, we have two ways to bold element that yield the exact same result. The Bold (<b></b>) and strong (<strong></strong>) element are the two ways. In most cases, when we bold text, we want to emphasize or stress the importance of what we are bolding. In this case, we use the semantic strong element. It is semantic because it lets us know that whatever is inside the strong element is important. The bold element is used more if we want to bold most of the text on our website. Generally you can use either or but it is recommended to use the semantic strong element if applicable. Here is an example of both elements in use:

As you can see, if we use the bold and strong elements on a paragaraph element, they look exactly the same. Note that header element (h1, h2, h3) are automatically bolded so you will not notice a difference if you use them. In your own program, write out an example using both the bold and strong element. Also note that is is best practice that we put the bold/strong elements inside of the paragraph element and not outside. Let's move onto the next modifier.

Emphasis & Italic Elements

In HTML, we have two ways to italicize text that yield the same result. The italic (<i></i>) and emphasis (<em></em>) element are the two ways. The italic element is usually used to denote text that is unusual such as a different language as well as references such as a book name. The emphasis element is used to stress importance and emphasize a word or phrase in a sentence. The emphasis element is semantic, meaning when we see the em tag, we know the text inside is important, whereas with the italic element, it could simply be a book title. Speaking of book titles, whenever we are referencing an artistical work, it is best to use the cite element (<em></em>), which is exactly the same as the italic element but it is semantic. Underline & Ins Elements

The Underline element (<u></u>) simply underlines the text. It is used to stress importance. However, it is key to know that when using an anchor tag, the underline property is added automatically. The Ins element (<ins></ins>) is functionally the same as the underline element but it is semantic. It indicates that text has been inserted. Strikeout & Del Elementa

The Strikeout element (<strike></strike> or <s></s> or <del></del>) adds a line through the center of text. strike or s is used to denote that something is old/defunct/out of order or that something has been canceled. There is no difference between using strike or s. The s element is simply an abbreviation. Personally, I would use the short s. Please note that both can be interchanged. However, the del element, although functionally the same, is semantically different and denotes something that has been removed. Here is an example of them in play:

Superscript & Subscript Elements

The superscript and subscript are two elements that allow us to format our characters in very specific situations. Superscripts (<sup></sup>) constrict our text to the top half of the line, we often use them in mathematics and physics so if you have a website centered around either of those, you will need this element. Superscripts can also be used to denote a clause or definition that would then be put at the bottom of the page (fine print). Subscripts (<sub></sub>) constict our text to the bottom half of the line and are commonly used in Physics, Mathematics and Chemistry as well as definitions. Here are both in play:

As you can see, both have their seperate uses. In most cases, we would not superscript or subscript a full sentence. Usually they are used to write formulas like H2O. It is important to NEVER use superscripts or subscripts to make font smaller, we use CSS for that. Next Element!

Bdo Element

The bdo element (<bdo></bdo>), which stands for Bi-directional overide, simply allows you to choose which direction your text should be in. It has a 'dir' attribute which stands for direction and has to possible option. ltr (left to right) or rtl (right to left). By default, it is set to ltf, which is how most text is read. However, some languages such as arabic are read right to left so the bdo element needs to be used. Center Element

The center element (<center></center>) allows us to center an element inside of it's container (if it does not have a container then the body element is the container). It is not recommended for use because CSS can do the same thing and gives us many more options on top of that.

Font Element

The font element (<font></font>) is an elemnt with attributes that allow us to change the color (<font></font>), font family (<font></font>) and size (<font size="1"></font>) of our text. The color attribute takes any color, hexcode, or rgb value. The font family attribute takes the name of a font-family such as 'arial'. The size attribute takes any number between one and seven, with one being the smallest text size and seven being largest.
