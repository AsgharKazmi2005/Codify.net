Learn HTML

Welcome to the HTML Portal. Here you can find everything you need to learn the basic fundamentals of HTML. HTML stands for Hypertext Markup Language, which may seem like nonsense at first. But really, this HTML powers everything your looking at right now. You know, all those sites like Facebook, YouTube, Twitter, Tiktok, Instagram, rely on HTML to function. Every Facebook Post, Youtube Title, Tweet, Story, Video, Message, almost everthing requires HTML to work. Let's get ready to learn HTML. To do so we need to Set-up our coding Environment. A 'coding environment' is essentially where you will write your code and execute it. At the moment we will execute our code locally, we will learn to launch a real site further down the line. Let's get into the Set-up.

VSCode Set-Up

The way you Set-Up will depend on your system. If your device is capable of downloading and running programs, continue reading, if not, click here. To execute our code, we are going to use VSCode, which is one of the most popular code editors out there. You can download it for macOS, Windows, or Linux here. Before we open VSCode, we are going to create a text file in our PC and name it index.html. 'index.html' is the common name for HTML files and every HTML file you make should be called index.html. The next step is to complete our download and open VSCode. Once opened, click the 'Open File' option and select the index.html file we created earlier. At this point, your VSCode should look something like this (Your layout may look different):

Cool, you have most of your coding environment set-up. But if you look closely, in the bottom right of my screen, it says "Go Live", an option you do not have. Don't worry it is not too complicated. What you want to do it navigate to the extensions section. We can use the CTRL + SHIFT + X shortcut to get there. Once there, we want to search up 'live server' in the search bar. We are looking for the extension with a purple icon made by Ritwick Dey, click on the extension and you should have a screen that looks like this:

Great, now all we have to do is click install and we are done with our setup. live-server creates a local web server so we can put our code online and actually see it. You can close the extension tab and return to your index.html file. You should see a 'Go Live' button in the bottom right now. If you click it, you will see a blank browser page open up, if so, then you have setup properly, you can now move onto the next section.

Repl.it Set-Up

If your device is incapable of download and running programs, don't worry, there are other ways to write our code. We can use Repl.it, which is a website that compiles the important HTML services and allows you to use them without downloading a thing. To set-up, go to Repl.it and sign up. Once you have signed up and reached your dashboard, go to the top left and click the 'Create +' button. You will now see a modal pop-up asking you which template you would like to use. We will be using the HTML, CSS, JS (Auto-Refresh) template. So search up HTML and make sure you click the template with Auto-Refresh because that will make our life easier. Title your Repl anything you want and 'Create Repl'. Your screen should look like so:

Brilliant, you have now set-up Repl for our HTML code. Be aware that Repl is only a minified version and lacks some features that an installation would have. Be sure to download VSCode on another device when you have the chance as VSCode is a professional coding environment. Even professional web designers use VSCode to write their code, but for now, Repl should be more than enough!
